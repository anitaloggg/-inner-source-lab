   # Global-UI 組件庫

   ## 🚀 專案簡介  
   Global-UI 是 Global-Tech Solutions 的官方 UI 框架，旨在統一公司內所有產品的視覺風格與交互經驗，減少重複開發。

   ## 🛠 快速上手  
   1. 安裝套件：`npm install global-ui`  
   2. 啟動開發伺服器：`npm start`
