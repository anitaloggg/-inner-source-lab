# Inner Source Lab 行為準則 (Code of Conduct)

就像整個技術社群一樣，Inner Source Lab 團隊與社群是由來自世界各地、涵蓋專業人士與志工的多元夥伴所組成。我們共同致力於完成各項任務，包括導師指導（Mentorship）、教學以及連結人才。

**多樣性**是我們強大的優勢，但有時也可能導致溝通問題或不愉快。為此，我們制定了一些基本規則，要求大家共同遵守。本準則一律適用於創辦人、導師以及尋求幫助與指引的人員。

這並非一份「禁止清單」，請將其視為一份指南，旨在讓我們每個人以及我們參與的技術社群變得更加豐富。

本行為準則適用於所有由 Inner Source Lab 專案管理的空間，包括 IRC、郵件列表（Mailing lists）、問題追蹤器（Issue tracker）、DSF 活動，以及任何由專案團隊建立供社群溝通的論壇。此外，在這些空間之外的違規行為，也可能影響該成員在社群內的參與權限。

如果您認為有人違反了行為準則，請發送郵件至 [huichen.lo@gmail.com](mailto:huichen.lo@gmail.com) 檢舉。

---

### 基本原則

* **保持友善與耐心。**
* **展現歡迎態度。** 我們致力於建立一個歡迎並支持各種背景與身份的社群。這包括但不限於：種族、族裔、文化、國籍、膚色、移民身份、社會與經濟階層、教育程度、性別、性傾向、性別認同與表達、年齡、身型、家庭狀態、政治信仰、宗教，以及心理與生理能力。
* **體貼他人。** 您的成果會被他人使用，而您也會依賴他人的工作成果。您所做的任何決定都會影響到使用者與同事，因此在做決策時應考慮其後果。請記住，我們是一個全球性的社群，您所使用的語言可能不是他人的母語。
* **保持尊重。** 我們不可能隨時都達成共識，但意見分歧不能作為行為不端或失禮的藉口。每個人都可能偶爾感到沮喪，但我們不能讓這種挫折感演變成個人攻擊
